# ai-hub/docs — Architecture & Reference Docs

General documentation, architecture diagrams, system overviews, and technical references.

## What goes here

- **Architecture diagrams** — system design, data flows, component relationships
- **API reference** — endpoints, schemas, event formats
- **Technical guides** — how to run tests, deploy, debug
- **System overviews** — high-level descriptions of major components
- **Design documents** — design rationale for major decisions

## Current structure

### `architecture/`
Two complementary diagrams:
- **architecture.md (Mermaid)** — System flow: how proof submissions and map requests flow through the bot (data flow diagram)
- **ai-hub-architecture.c4 (LikeC4)** — Folder structure: where everything lives in ai-hub/ (organizational/navigation diagram)
