# ai-hub/research — Data-Driven Analysis & Exploration

Research output, investigations, experiments, and findings on specific topics.

## What goes here

- **Model analysis** — evaluations of ML models, architecture comparisons
- **Performance investigations** — profiling, benchmarking, optimization studies
- **Data gathering** — analysis of logs, metrics, user behavior
- **Feasibility studies** — "Can we do X?" research, proof-of-concept findings
- **Topic-specific folders** — organized by area (e.g., `proof-detection-model/`)

## Current structure

- `proof-detection-model/` — ML model training notebooks, architecture studies
- `proof_system_research.md` — Analysis of proof workflow and system design
